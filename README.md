# rolfcopter

Nothing is more fun that watching a roflcopter fly!

![FLY](img/roflcopter.gif "ROFLCHOPTER")

## Compilation

```bash
mkdir build
cd build
cmake ../
make
```

## Installation
```bash
sudo make install
```

## Create RPM
```bash
make package
```
