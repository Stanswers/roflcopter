# rolfcopter

Nothing is more fun that watching a roflcopter fly!

![FLY](img/roflcopter.gif "ROFLCHOPTER")

## Compilation

```bash
make
```

## Installation
To be executed as root.
```bash
make install
```

## Uninstall
To be executed as root.
```bash
make uninstall
```
